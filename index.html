FindR

FindR (Chrome Extension)

My first software project; a Chrome extension that offers full find and replace capabilities for Google Chrome (description from the extension store below). It offers several options, including regular expression support for searching. Written using HTML5, CSS3, JQuery, and the Material Design Lite framework.

Over 100 users!

Find and replace is a handy extension that does exactly what the name implies: it finds and replaces words, allowing you to completely change the text of any website. On a more practical note, it also has an input field option that allows you to easily find and replace all the text in input fields (such as forms, etc.)

Recyclo

Recyclo: An Educational Recycling Game

At the first annual Clements High School Hackathon, within roughly 12 hours, my team of three developed an educational game and website designed to teach citizens about recycling and how to properly recycle various types of materials. I was the main website developer and created the website from scratch using the Bootstrap framework and the GSAP framework for JS animation. We placed second overall.
